# TravelGlobe iOS App

Open `TravelGlobe.xcodeproj` in Xcode, select an iPhone simulator, then press Run.

If you run on a real iPhone, open Signing & Capabilities and select your Apple Developer Team.

Features:
- SwiftUI + SceneKit 3D globe
- Drag to rotate, pinch to zoom
- Tap glowing pins to view travel records
- Add places by city name using built-in coordinates
- Delete records
- UserDefaults local persistence
- Responsive iPhone layout
